## 说明

图形化CTA策略回测模块，基于用户友好的图形界面实现简洁易用的数据下载、历史回测和参数优化等投研功能。

## 安装

安装环境推荐基于3.9.0版本以上的【[**VeighNa Studio**](https://www.vnpy.com)】。

直接使用pip命令：

```
pip install vnpy_ctabacktester
```


或者下载源代码后，解压后在cmd中运行：

```
pip install .
```
